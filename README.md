UbuntuBot

UbuntuBot is a simple AI-powered chatbot that provides legal advice. This project is built with a Node.js backend and a Vite-powered React frontend.

Features

AI Chatbot: Interact with UbuntuBot for instant legal advice.

User Registration: Sign up and manage your account.

Simple and Clean UI: Easy-to-navigate frontend.

Getting Started
Prerequisites

Node.js (v18+)

NPM or Yarn

Installation

Clone the repository:

git clone https://github.com/yourusername/ubuntu-bot.git


Navigate to the project directory:

cd ubuntu-bot


Install backend dependencies:

cd backend
npm install


Install frontend dependencies:

cd ../frontend
npm install

Running the Project

Start the backend:

cd backend
npm start


Start the frontend:

cd ../frontend
npm run dev


Access the app:
Open your browser and go to http://localhost:5173.

Usage

Sign Up: Register a new account to start interacting with UbuntuBot.

Chat: Once logged in, you can ask legal questions and receive AI-powered responses.

Technologies Used

Frontend: React, Vite

Backend: Node.js, Express, OpenAI API

Styling: CSS
